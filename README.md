# Machine Learning Based Lightweight Intrusion Detection System for IoT

## Overview
This project implements a **Lightweight Intrusion Detection System (IDS)** for the **Internet of Things (IoT)** using **Machine Learning techniques**.  
It is designed to efficiently detect and classify network intrusions while minimizing computational and memory usage â€” ideal for IoT devices.

---

## Objective
To develop a **supervised ML-based intrusion detection model** capable of identifying various types of cyber-attacks such as **Denial of Service (DoS)**, **Data Injection**, and **Malicious Access** with high accuracy and low latency.

---

## Technologies Used
- **Language:** Python
- **Framework:** Tkinter (for GUI interface) 
- **Libraries:**  
  - `numpy`  
  - `pandas`  
  - `matplotlib`  
  - `scikit-learn`  
  - `tkinter`  
- **Platform/IDE:** VS Code / PyCharm / Jupyter Notebook  
- **Operating System:** Windows 8 or above  

---

## Algorithms Implemented
- Support Vector Machine (SVM)  
- Decision Tree  
- Logistic Regression  
- K-Nearest Neighbors (KNN)  
- Naive Bayes  
- Random Forest  
- Multi-Layer Perceptron (MLP)

---

## Output
- Displays **accuracy** of each algorithm  
- Generates **attack-type** and **accuracy comparison** graphs  
- Provides **visual insights** on intrusion detection performance
